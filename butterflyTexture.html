3D Shaders

I wanted to try texturing a 3D object, so I used p5.js to make a simple rotating cube and apply a pattern to it. I picked trigonometric equations that created a mosaic style pattern that looked kind of like a butterfly to me, and then had the color change with the variables in the equations. Access it in the p5.js web editor here. p5.js
