Camera face filters

For this project, I was interested in facial recognition for camera filters and even 3d model rigging. I originally starting with ml5 (to create the thumbnail), and ended up using MediaPipe Facemesh in p5.js to recognize 3d facial landmarks and make a few simple filters in (use arrow keys to switch). The end result is a simple camera filter that can serve as a baseline for more advanced face filters, like those seen on Snapchat, TikTok, etc. Access it in the p5.js web editor here. p5.js
