Solar System Generator

A simple, randomly generated solar system poster with modifiable attributes (background star density, sun size, planet #). Hide and show the slider text using the left and right arrow keys. Access it in the p5.js web editor here. p5.js

I have recently been interested in facial recognition for camera filters and even 3d model rigging. For this project, I originally starting with ml5 (to create the thumbnail), and ended up using MediaPipe Facemesh in p5.js to recognize 3d facial landmarks and make a few simple filters in (use arrow keys to switch).
